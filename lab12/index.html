Lab Summary

Create a function that includes using conditionals.

Problems

I wanted my function to return unique text depending on the results, and not just fill in a blank with a variable. I knew I had to use something else instead of "return", but I was lost for a long time on what I can use to change text. Then, I remembered that I can use if-elses to change the value of the same variable. In this case I made the variable be the "output" div itself.
